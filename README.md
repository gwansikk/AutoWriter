## Celpilot

양식파일과 데이터파일을 통해서 반복적인 문서작업을 자동화하여 쉽게 파일을 만들어 줍니다.

| OS      | STABLE | BETA  |
| ------- | ------ | ----- |
| macOS   | -      | Ready |
| Windows | -      | -     |

## Installation

1. 필요한 라이브러리를 설치합니다. (`pip install -r requirements.txt`)
2. `main.py`를 실행합니다.
